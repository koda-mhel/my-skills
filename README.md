# my-skills

A collection of personal Codex skills.

---

## Installation

### Method 1: Using skill-installer (Recommended)

1. Open codex
2. Enter the following command:

```
$skill-installer install https://github.com/koda-mhel/my-skills/tree/main/skills/{skill-name}

# Example:
$skill-installer install https://github.com/koda-mhel/my-skills/tree/main/skills/flutter-page-scaffolder
```

3. Restart codex after installation completes
4. The skill will be available for use

---

### Method 2: Manual Installation

1. Clone this repository:

```bash
git clone https://github.com/koda-mhel/my-skills.git
```

2. Copy the desired skill folder into your local skills directory:

```bash
cp -r my-skills/skills/{skill-name} ~/.codex/skills/
```

3. Restart codex
